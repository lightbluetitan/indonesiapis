Licensing Information for IndonesiAPIs Package

This package, IndonesiAPIs, includes datasets sourced from various R packages. In order to maintain consistency and clarity, a naming convention has been applied where suffixes are added to the dataset names to indicate the type and format of the datasets. The original content of these datasets has not been altered in any way.

Dataset Licensing Information:

- indonesia_latlong_df:
  Originally Loc from the gstar 0.1.0 package, licensed under GPL-2 | GPL-3. The dataset name has been changed to 'indonesia_latlong_df' to indicate that this dataset is a data frame object.

- poverty_index_tbl_df:
  Originally ppiIDN2023 from the ppitables 0.6.0 package, licensed under MIT + file LICENSE. The dataset name has been changed to 'poverty_index_tbl_df' to indicate that this dataset is a tibble object.

- Bali_tbl_df:
  Originally Bali from the ifpd 0.1.0 package, licensed under GPL-3. The dataset name has been changed to 'Bali_tbl_df' to indicate that this dataset is a tibble object.

- SumateraUtara_tbl_df:
  Originally SumateraUtara from the ifpd 0.1.0 package, licensed under GPL-3. The dataset name has been changed to 'SumateraUtara_tbl_df' to indicate that this dataset is a tibble object.

- DKIJakarta_tbl_df:
  Originally DKIJakarta from the ifpd 0.1.0 package, licensed under GPL-3. The dataset name has been changed to 'DKIJakarta_tbl_df' to indicate that this dataset is a tibble object.
